# HackerBot for Slack
## An extremely basic Slack bot for Saying Something Smart™️.

## Installation
1. Clone the repo:
  `git clone git://blahblahblah`
1. Bundle:
  `bundle install`
1. Go!
  `bundle exec rackup`
